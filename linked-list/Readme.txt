Purpose of the software is to create a single linked list and helpful functions to make changes as "deleteNode", "searchNode", "printList" and others. 

- There are 2 c files: main.c and linked_list.c and header file for linked_list.c
- Compile: gcc main.c linked_list.c -o a3
- Run: ./a3
